# My TypeScript Full-Stack Project

โปรเจกต์ตัวอย่างใช้ **TypeScript + Express + Prisma + SQLite** สำหรับ backend และ API CRUD ของ `User`

---

## Features

- REST API CRUD สำหรับ User
- ใช้ **Prisma** ทำงานกับ SQLite
- ใช้ **TypeScript** สำหรับ backend
- สามารถทดสอบ API ด้วย curl หรือ Prisma Studio
- เตรียมพร้อมสำหรับการต่อยอด frontend (React, Vue, หรือ Next.js)
